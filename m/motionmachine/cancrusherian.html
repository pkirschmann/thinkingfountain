Can Scruncher (Neat and Upbeat)

By: Ian

My machine is a can scruncher. It scrunches cans when you don't want to stomp on them. I built my machine with two pieces of wood and some kind of hinge. If I were to improve my machine I would make an automatic start and scruncher.

Troubleshooting:

When I made my machine there was a slight problem with the can flying out of the scruncher. I fixed it by using and extra piece of scrap wood.
