Mountain Lift (a toy or a game)

By: Emily

My machine is either a model or a toy of a mountain lift. If you pull on the string it will move the box. If you turn the ribbon holders it will move the box. If you push it it will move back and fourth. If you pull on the white string it will move.

My machine started out to be a small cleaning device. Then it was a car that was moved by a fan. Now it is a mountain lift type thing. We turned it into a mountain lift by flipping my car upside down. It is made of beads,a card board box, string, string holders, wood, card board paper towel roll cut in half, nails, hanger and some team work.

If I could change it I would use plastic string holders.
